Up: [C Structures](c_structures)
Brother(s):

Structures are defined like this:

Example: Struct for employee

```C
struct employee
{
	int ID;
	int age;
	int hoursPerDay;
};
```

And more generally:

```C
struct structName
{
	type member1;
	type member2;
	type member3;
	.
	.
};
```
