There are two main categories of types in XSD: 
	- SimpleType:
		- Contains only text
		- Contains no further elements
		- Cannot be empty
		- Cannot have attributes
	- ComplexType: 
		- Can have attributes
		- Can have children (can contain other elements)
		- Represents custom types
