Tuples are like lists, except they cannot be changed. The slot values cannot change, the length cannot change

<hr>

## Create tuple by literal expression

- ()
	- Empty tuple
- (_a,_)
	- tuple with 1 item 
- (_a, b, c_)
	- tuple with item _a, b, c_

## Convert iterable to tuple

	tuple(iterable)
