For loops are a **count-controlled** loop that stop if the counter value reaches its limit.

**Java representation:**

```java
public void forLoop() {
	for (int i; i < 6; i++) {
		// do something
	}
}
```

For loops stop once the condition (`i < 6` above) is met, and the looping code does not run once this happens.
