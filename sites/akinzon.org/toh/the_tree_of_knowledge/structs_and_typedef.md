Typedef an be used on structures in two ways:

The intuitive way:

```c
typedef struct employee worker;
```

The compact way, where the typedef is used in the [C Structure Definition|definition](c_structure_definition|definition):

```C
typedef struct 
{
	int ID;
	int age;
	int hoursPerDay;
} worker;
```
